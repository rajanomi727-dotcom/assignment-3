## 🚀 Getting Started

Follow these steps to explore the projects on your local machine:

### 1. Install Dependencies

Ensure you have Node.js installed, then run the following command in the root directory:

```bash
npm install
```

### 2. Start the Development Server

Launch the Expo server to run the applications:

```bash
npx expo start
```

From the Expo CLI, you can open the apps on:
- A physical device using the [Expo Go](https://expo.dev/go) app.
- An [Android emulator](https://docs.expo.dev/workflow/android-studio-emulator/).
- An [iOS simulator](https://docs.expo.dev/workflow/ios-simulator/).
